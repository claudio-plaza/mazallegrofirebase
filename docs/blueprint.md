# **App Name**: ClubZenith

## Core Features:

- Role-Based Authentication: User authentication with different roles (socio, portero, medico, administrador) based on simulated user data.
- Personalized Dashboard: Dashboard displaying relevant information and quick access features based on the user's role.
- Digital Carnet: Digital membership card displaying user information and medical fitness status.
- New Member Registration: Multi-step form for new member registration with detailed data input and validation.
- Medical Panel: Medical panel to manage member medical reviews and fitness status, with quick overviews.
- Access Control: Access control interface to verify member status and medical clearance.
- AI-Driven Member Status Tool: Generative AI powered recommendation tool for admin users: suggest updates or changes to the status of club members, for review. For example, flagging members with expired medical evaluations or recommending account deactivation.

## Style Guidelines:

- Primary color: Saturated blue (#3498db) for a sense of trustworthiness and vibrancy.
- Background color: Light gray (#f0f2f5), almost white to provide a clean and modern feel.
- Accent color: Purple (#9b59b6) to draw attention to interactive elements.
- Clean and modern typography with good readability for all user roles.
- Use of simple and clear icons to represent different actions and sections within the application.
- Clean and intuitive layout, making navigation easy for users with different access levels.
